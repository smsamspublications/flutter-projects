# Dart & Flutter — A Complete Language & Framework Manual

A comprehensive, 138-page Dart and Flutter reference manual, written as structured textbook chapters with a quick-reference appendix section — created for the **JNTUK B.Tech AI & DS curriculum (R23 Regulation)**.

📘 **[Download the manual (.docx)](./Dart_Flutter_Complete_Manual.docx)**

---

## What's Inside

- **Part I — The Dart Language**
  Variables, control flow, functions, collections, OOP, generics, exception handling, async/await & Streams, null safety, packages, extension methods, records & pattern matching, isolates.

- **Part II — The Flutter Framework**
  Widgets, layouts, navigation, state management (`setState` → `Provider` → `Riverpod` → `Bloc`), networking, local persistence, animations, theming, accessibility, internationalization, Firebase, platform channels, performance optimization, and deployment.

- **Part III — Capstone Project**
  A complete, 9-step walkthrough building a working "Faculty Activity Tracker" app end-to-end — model, repository, state layer, UI, and tests.

- **Part IV — JNTUK R23 Lab Manual**
  All 10 prescribed Flutter Application Development lab experiments, each with:
  - Aim & software required
  - Fully line-commented, runnable programs for both sub-parts (a) and (b)
  - Expected output
  - Result statement
  - 5 viva-voce questions per experiment

- **Part V — Quick-Reference Appendices**
  Dart syntax cheat sheet, Flutter widget catalog, troubleshooting guide, package directory, CLI/DevTools reference, glossary, practice/viva questions, and suggested mini-project ideas.

---

## How to Use This Manual

Open the `.docx` file in Microsoft Word. The Table of Contents is auto-generated — right-click it and choose **Update Field** (or press **F9**) once to populate page numbers.
